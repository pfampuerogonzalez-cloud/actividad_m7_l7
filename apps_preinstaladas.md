# 1. ¿QUÉ SON LAS APLICACIONES PREINSTALADAS?

### UNA APLICACIÓN PREINSTALADA EN DJANGO ES UN MODULO DE CODIGO INTEGRADO QUE VIENE ACTIVADO POR DEFECTO EN LA LISTA ```INSTALLED_APPS``` DENTRO DEL ARCHIVO DE CONFIGURACIÓN ```SETTINGS.PY``` EN EL PROYECTO.

---

## BLOQUE INSTALLED APPS

- ```django.contrib.admin```: PANEL DE ADMINISTRACIÓN AUTOMÁTICO PARA GESTIONAR LA DB.

- ```django.contrib.auth```: SISTEMA COMPLETO DE AUTENTIFICACIÓN (USUARIOS, GRUPOS, PERMISOS Y CONTRASEÑAS).

- ```django.contrib.contenttypes```: FRAMEWORK QUE PERMITE RASTREAR Y RELACIONAR TODOS LOS MODELOS DE LA DB

- ```django.contrib.sessions```: SISTEMA PARA GESTIONAR SESIONES DE USUARIO DE FORMA ANÓNIMA O AUTENTICADA.

- ```django.contrib.messages ```: FRAMEWORK DE MENSAJERÍA PARA MOSTRAR NOTIFICACIONES TEMPORALES AL USUARIO.

- ```django.contrib.staticfiles ```: HERRAMIENTA ENCARGADA DE GESTIONAR Y SERVIR ARCHIVOS ESTÁTICOS COMO IMÁGENES, HOJAS DE ESTILOS CSS Y JAVASCRIPT.
